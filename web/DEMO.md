# 推荐问题功能演示

## 快速体验

### 1. 启动应用
```bash
cd web
npm run dev
```

### 2. 访问聊天页面
在浏览器中打开：`http://localhost:3000/chat`

### 3. 查看推荐问题
页面加载后，你会看到：
- 一个灯泡图标（带脉冲动画）
- "推荐问题"标题
- 8 个精美的问题卡片

### 4. 交互体验
- **悬停效果**：鼠标悬停在任意卡片上
  - 卡片上浮
  - 顶部出现蓝色渐变条
  - 图标放大并旋转
  - 右侧箭头滑入
  
- **点击发送**：点击任意卡片
  - 自动发送预设问题
  - 推荐问题区域消失
  - AI 开始回复

### 5. 重新显示
- 刷新页面
- 或清空所有对话记录
- 推荐问题会重新显示

## 动画效果时间轴

### 页面加载（0-600ms）
```
0ms    ─→ 推荐问题区域开始淡入
100ms  ─→ 标题和图标显示
200ms  ─→ 问题卡片开始显示
600ms  ─→ 动画完成
```

### 悬停效果（0-300ms）
```
0ms    ─→ 鼠标悬停
50ms   ─→ 顶部渐变条展开
100ms  ─→ 卡片开始上浮
150ms  ─→ 图标开始放大旋转
200ms  ─→ 箭头滑入
300ms  ─→ 动画完成
```

### 点击发送（0-1500ms）
```
0ms    ─→ 点击卡片
50ms   ─→ 推荐问题淡出
100ms  ─→ 用户消息显示
200ms  ─→ 加载动画开始
1500ms ─→ AI 回复显示
```

## 常见问题

### Q: 推荐问题不显示？
A: 检查是否有历史消息，刷新页面清空消息。

### Q: 动画不流畅？
A: 检查浏览器版本，建议使用最新版 Chrome。

### Q: 移动端显示异常？
A: 清除浏览器缓存，重新加载页面。

### Q: 如何自定义问题？
A: 编辑 `ChatContainer.vue` 中的 `recommendedQuestions` 数组。
