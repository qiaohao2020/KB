# QuickSearch Web - 功能说明

## 新增功能：推荐问题

### 功能概述
在聊天页面添加了推荐问题功能，当用户首次进入或清空对话后，会显示精心设计的推荐问题卡片，帮助用户快速开始对话。

### 功能特点

#### 1. 智能显示
- 当聊天记录为空时自动显示推荐问题
- 用户发送第一条消息后，推荐问题自动隐藏
- 清空对话后，推荐问题重新显示

#### 2. 精选问题
提供 8 个精心设计的推荐问题，涵盖常见使用场景：

1. **代码优化** 🔧
   - 图标：代码符号
   - 描述：如何优化这段代码的性能？
   - 适用场景：代码性能优化、代码审查

2. **学习建议** 📚
   - 图标：书本
   - 描述：如何快速学习一门新技术？
   - 适用场景：技术学习、职业发展

3. **问题排查** 🐛
   - 图标：Bug
   - 描述：帮我分析这个错误的原因
   - 适用场景：问题调试、错误分析

4. **创意灵感** 💡
   - 图标：灯泡
   - 描述：给我一些项目创意
   - 适用场景：项目规划、创意头脑风暴

5. **文档编写** 📝
   - 图标：文件
   - 描述：如何写好技术文档？
   - 适用场景：文档撰写、技术写作

6. **性能分析** 📈
   - 图标：图表
   - 描述：分析应用性能瓶颈
   - 适用场景：性能优化、系统分析

7. **安全审查** 🛡️
   - 图标：盾牌
   - 描述：检查代码安全问题
   - 适用场景：安全审计、代码审查

8. **设计建议** 🎨
   - 图标：调色板
   - 描述：改善用户界面设计
   - 适用场景：UI/UX 设计、界面优化

### 使用方法

1. **访问聊天页面**
   - 打开应用后点击"聊天"菜单
   - 或直接访问 `/chat` 路由

2. **选择推荐问题**
   - 浏览 8 个推荐问题卡片
   - 点击任意卡片快速开始对话
   - 系统会自动发送预设的问题

3. **开始对话**
   - 点击问题后立即发送
   - AI 开始处理并回复
   - 推荐问题自动隐藏

### 技术实现

#### 组件结构
```vue
<div class="recommended-questions">
  <div class="recommended-header">
    <!-- 标题和说明 -->
  </div>
  <div class="questions-grid">
    <div class="question-card">
      <!-- 问题卡片 -->
    </div>
  </div>
</div>
```

#### 数据结构
```javascript
{
  id: 1,
  icon: 'bi bi-code-slash',      // Bootstrap Icons 图标类
  title: '代码优化',              // 问题标题
  description: '如何优化...',     // 问题描述
  question: '我有一段代码...'     // 实际发送的问题
}
```

#### 显示逻辑
```javascript
// 当消息数组为空时显示推荐问题
v-if="messages.length === 0"
```

### 自定义配置

#### 修改推荐问题
编辑 `web/src/components/ChatContainer.vue` 文件：

```javascript
const recommendedQuestions = ref([
  {
    id: 1,
    icon: 'bi bi-your-icon',      // 修改图标
    title: '你的标题',             // 修改标题
    description: '你的描述',       // 修改描述
    question: '你的问题内容'       // 修改实际问题
  },
  // 添加更多问题...
])
```

#### 修改显示条件
```javascript
// 当消息少于 3 条时显示
v-if="messages.length < 3"

// 始终显示
v-if="true"

// 根据其他条件显示
v-if="showRecommended && messages.length === 0"
```

#### 修改样式
在 `<style scoped>` 部分修改：

```css
/* 修改卡片颜色 */
.question-card {
  border-color: #your-color;
}

/* 修改图标背景 */
.question-icon {
  background: linear-gradient(135deg, #color1, #color2);
}

/* 修改网格列数 */
.questions-grid {
  grid-template-columns: repeat(3, 1fr); /* 固定 3 列 */
}
```
