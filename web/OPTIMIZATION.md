# 推荐问题优化说明

## 问题描述
在悬浮推荐问题卡片时，由于卡片的阴影和动画效果超出了容器边界，导致出现横向滚动条，影响用户体验。

## 优化方案

### 1. 尺寸优化

#### 卡片尺寸
| 属性 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 最小宽度 | 280px | 240px | -40px |
| 内边距 | 20px | 16px | -4px |
| 间距 | 16px | 12px | -4px |

#### 图标尺寸
| 属性 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 图标容器 | 48px | 40px | -8px |
| 图标字体 | 24px | 20px | -4px |
| 圆角 | 12px | 10px | -2px |

#### 文字尺寸
| 属性 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 标题字体 | 16px | 15px | -1px |
| 描述字体 | 13px | 12px | -1px |
| 行高 | 1.5 | 1.4 | -0.1 |

#### 箭头尺寸
| 属性 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 箭头容器 | 32px | 28px | -4px |
| 箭头图标 | 16px | 14px | -2px |
| 圆角 | 8px | 6px | -2px |

### 2. 动画优化

#### 悬浮效果
| 效果 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 上浮距离 | 4px | 2px | -50% |
| 阴影范围 | 0 12px 24px | 0 8px 16px | -33% |
| 阴影透明度 | 0.15 | 0.12 | -20% |

#### 图标动画
| 效果 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 缩放比例 | 1.1 | 1.05 | -50% |
| 旋转角度 | 5deg | 3deg | -40% |

#### 箭头动画
| 效果 | 优化前 | 优化后 | 变化 |
|------|--------|--------|------|
| 移动距离 | 4px | 2px | -50% |

### 3. 容器优化

#### 添加的属性
```css
/* 聊天消息容器 */
.chat-messages {
  overflow-x: hidden;  /* 防止横向滚动 */
}

/* 推荐问题容器 */
.recommended-questions {
  max-width: 100%;     /* 限制最大宽度 */
  overflow: hidden;    /* 隐藏溢出内容 */
}

/* 网格容器 */
.questions-grid {
  max-width: 100%;     /* 限制最大宽度 */
  padding: 0 4px;      /* 减小内边距 */
}

/* 卡片 */
.question-card {
  will-change: transform;  /* 优化动画性能 */
}
```

### 4. 响应式优化

#### 移动端调整
| 属性 | 桌面端 | 移动端 | 说明 |
|------|--------|--------|------|
| 网格列数 | auto-fill | 1 | 单列布局 |
| 卡片内边距 | 16px | 14px | 更紧凑 |
| 图标尺寸 | 40px | 36px | 更小巧 |
| 间距 | 12px | 10px | 更紧凑 |
| 悬浮上浮 | 2px | 1px | 更轻微 |
| 阴影范围 | 0 8px 16px | 0 4px 12px | 更柔和 |
