# Vue浏览器扩展加载指南

## 🚀 如何加载Vue浏览器扩展

### 步骤1：构建项目
```bash
npm run build
```

### 步骤2：在Chrome浏览器中加载扩展

1. **打开Chrome扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 在页面右上角打开"开发者模式"开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择项目的 `dist` 文件夹
   - 点击"选择文件夹"

### 步骤3：使用扩展

1. **查看扩展图标**
   - 扩展加载成功后，会在浏览器工具栏显示QuickSearch图标

2. **打开Popup**
   - 点击扩展图标打开popup窗口
   - 可以通过popup快速访问功能

3. **打开侧边栏**
   - 点击popup中的"打开侧边栏"按钮
   - 或使用快速操作按钮

4. **导航功能**
   - 知识库：管理和浏览知识内容
   - 内容检索：搜索相关信息
   - 智能问答：AI聊天功能
   - 历史记录：查看使用历史
   - 设置：配置扩展选项

## 🔧 开发模式

### 实时开发
浏览器扩展不能使用 `npm run dev`，需要：

1. **修改代码后重新构建**
   ```bash
   npm run build
   ```

2. **刷新扩展**
   - 在 `chrome://extensions/` 页面
   - 点击扩展卡片上的刷新按钮
   - 或者重新加载扩展

### 调试技巧

1. **查看控制台**
   - Popup调试：右键popup → 检查
   - 侧边栏调试：F12打开开发者工具
   - 背景脚本：在扩展详情页点击"service worker"

2. **查看错误**
   - 扩展管理页面会显示错误信息
   - 控制台会显示JavaScript错误

## 📁 项目结构

```
dist/                          # 构建输出目录（加载此文件夹）
├── manifest.json             # 扩展清单文件
├── background.js             # 后台脚本
├── content.js               # 内容脚本
├── src/pages/
│   ├── popup/index.html     # Popup页面
│   ├── sidepanel/index.html # 侧边栏页面
│   └── options/index.html   # 选项页面
└── assets/
    ├── images/              # 图标和图片
    └── *.js, *.css         # 编译后的资源文件
```

## ⚠️ 常见问题

### 1. 清单文件错误
- 确保 `dist/manifest.json` 存在
- 检查JSON格式是否正确

### 2. 文件路径错误
- 确保所有引用的文件都在dist目录中
- 检查manifest.json中的路径是否正确

### 3. 权限问题
- 检查manifest.json中的permissions配置
- 确保host_permissions包含需要的域名

### 4. Vue Router问题
- 使用Hash模式适配扩展环境
- 确保路由配置正确
