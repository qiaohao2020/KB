# 智能问答预设Prompt功能

## 🎯 功能概述

为ChatContainer添加了预设Prompt功能，允许用户快速使用预定义的对话模式，并支持添加、编辑和管理自定义Prompt，提升对话效率和用户体验。

## ✨ 主要特性

### 1. **预设Prompt库**
- **内置模板**：5个常用的对话模式
- **快速启动**：一键发送预设内容
- **分类清晰**：涵盖代码、文档、问题分析等场景
- **即时可用**：无需配置即可使用

### 2. **自定义Prompt管理**
- **添加功能**：创建个性化的对话模板
- **编辑功能**：修改现有的自定义Prompt
- **删除功能**：移除不需要的自定义Prompt
- **持久化存储**：自动保存到localStorage

### 3. **智能界面设计**
- **可收缩面板**：节省屏幕空间
- **网格布局**：响应式卡片展示
- **悬停效果**：直观的交互反馈
- **状态记忆**：保持用户的显示偏好

### 4. **模态对话框**
- **现代化设计**：毛玻璃效果和圆角设计
- **表单验证**：确保输入内容的完整性
- **键盘友好**：支持ESC键关闭
- **响应式布局**：适配各种屏幕尺寸

## 🔧 技术实现

### 预设Prompt数据结构
```javascript
const prompts = ref([
  {
    id: 1,                    // 唯一标识
    name: '代码解释',         // 显示名称
    content: '请帮我解释...', // Prompt内容
    isDefault: true          // 是否为默认预设
  }
])
```

### 核心功能实现

#### 1. **使用Prompt**
```javascript
const usePrompt = (prompt) => {
  // 直接发送Prompt内容作为消息
  handleSendMessage(prompt.content)
  // 自动收起面板节省空间
  showPrompts.value = false
}
```

#### 2. **添加自定义Prompt**
```javascript
const savePrompt = () => {
  const newPrompt = {
    id: Date.now(),
    name: dialogData.value.name.trim(),
    content: dialogData.value.content.trim(),
    isDefault: false
  }
  prompts.value.push(newPrompt)
  savePromptsToStorage()
}
```

#### 3. **数据持久化**
```javascript
const savePromptsToStorage = () => {
  const customPrompts = prompts.value.filter(p => !p.isDefault)
  localStorage.setItem('quicksearch-custom-prompts', JSON.stringify(customPrompts))
}
```

#### 4. **状态管理**
```javascript
// 显示/隐藏状态持久化
localStorage.setItem('quicksearch-show-prompts', showPrompts.value.toString())

// 自定义Prompt持久化
localStorage.setItem('quicksearch-custom-prompts', JSON.stringify(customPrompts))
```

## 📋 内置Prompt模板

### 1. **代码解释**
```
请帮我解释这段代码的功能和工作原理，并指出可能的改进点。
```
- **适用场景**：代码审查、学习理解
- **使用方式**：粘贴代码后使用此Prompt

### 2. **文档总结**
```
请帮我总结这份文档的主要内容，提取关键信息和要点。
```
- **适用场景**：文档阅读、信息提取
- **使用方式**：提供文档内容后使用

### 3. **问题分析**
```
请帮我分析这个问题，提供解决方案和建议。
```
- **适用场景**：问题解决、决策支持
- **使用方式**：描述问题后使用此Prompt

### 4. **创意写作**
```
请以创意和有趣的方式帮我写作，注重文采和表达效果。
```
- **适用场景**：内容创作、文案写作
- **使用方式**：提供写作主题后使用

### 5. **学习辅导**
```
请以教师的角色帮我学习这个知识点，用简单易懂的方式解释。
```
- **适用场景**：知识学习、概念理解
- **使用方式**：提出学习问题后使用

## 🎮 交互功能

### 主要操作
1. **展开/收起**：点击标题区域的箭头按钮
2. **使用Prompt**：点击任意Prompt卡片
3. **添加Prompt**：点击"+"按钮打开对话框
4. **编辑Prompt**：点击卡片上的编辑按钮
5. **删除Prompt**：点击卡片上的删除按钮（仅自定义）

## 💾 数据管理

### 存储策略
- **默认Prompt**：硬编码在组件中，不可删除
- **自定义Prompt**：存储在localStorage中
- **显示状态**：记住用户的展开/收起偏好
- **数据格式**：JSON格式，便于序列化

### 数据结构
```json
{
  "quicksearch-custom-prompts": [
    {
      "id": 1640995200000,
      "name": "API文档生成",
      "content": "请帮我为这个API生成详细的文档...",
      "isDefault": false
    }
  ],
  "quicksearch-show-prompts": "true"
}
```

### 错误处理
- **加载失败**：使用默认Prompt继续工作
- **保存失败**：显示错误提示，不影响使用
- **数据损坏**：自动重置为默认状态
