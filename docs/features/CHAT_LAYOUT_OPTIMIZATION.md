# 智能问答页面布局优化

## 🎯 优化目标

优化智能问答页面的布局，让ChatContainer占用更多空间，以便显示更多聊天内容，提升用户的对话体验。

## 📏 空间分配优化

### 优化前后对比

| 区域 | 优化前 | 优化后 | 节省空间 |
|------|--------|--------|----------|
| 页面头部 | 较大 | 紧凑 | ~30px |
| 消息区域 | 标准 | 扩展 | +40px |
| 输入区域 | 较大 | 精简 | ~20px |
| 消息间距 | 24px | 16px | 8px/消息 |

### 总体效果
- **增加显示空间**：约90px额外空间用于显示聊天内容
- **提升内容密度**：可显示更多消息历史
- **保持美观性**：在节省空间的同时保持视觉美观

## 🔧 具体优化措施

### 1. **页面头部压缩**

#### 标题区域优化
```css
/* 优化前 */
.title-section h1 {
  font-size: 32px;
  margin: 0 0 8px 0;
}

/* 优化后 */
.title-section h1 {
  font-size: 24px;        /* 减小字体 */
  margin: 0 0 4px 0;      /* 减少间距 */
}
```

#### 内边距调整
```css
/* 优化前 */
.header-content {
  padding: 24px 32px;
}

/* 优化后 */
.header-content {
  padding: 16px 24px;     /* 减少内边距 */
}
```

#### 状态指示器精简
```css
/* 优化前 */
.status-indicator {
  padding: 8px 16px;
  font-size: 14px;
}

/* 优化后 */
.status-indicator {
  padding: 6px 12px;      /* 减少内边距 */
  font-size: 12px;        /* 减小字体 */
}
```

## 📱 响应式优化

### 移动端适配
```css
@media (max-width: 768px) {
  /* 头部区域 */
  .header-content {
    padding: 12px 16px;    /* 进一步减少移动端间距 */
  }
  
  /* 输入区域 */
  .chat-input {
    padding: 12px 16px;    /* 移动端紧凑布局 */
  }
  
  .message-textarea {
    padding: 10px 12px;    /* 减少文本区域间距 */
  }
  
  /* 页面主体 */
  .page-body {
    padding: 8px 12px 12px 12px;  /* 移动端最小间距 */
  }
}
```

### 字体大小调整
```css
@media (max-width: 768px) {
  .title-section h1 {
    font-size: 20px;       /* 移动端更小字体 */
  }
  
  .title-section p {
    font-size: 12px;       /* 描述文字更小 */
  }
  
  .status-indicator {
    font-size: 11px;       /* 状态指示更小 */
  }
}
```
