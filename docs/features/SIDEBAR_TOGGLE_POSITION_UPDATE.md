# 侧边栏Toggle按钮位置更新

## 🎯 更新内容

将侧边栏的收缩/展开按钮从右上角移动到侧边栏的垂直中间位置，提供更好的用户体验和视觉平衡。

## 🔧 技术实现

### CSS 更改

#### 主要定位更新
```css
.toggle-btn {
  position: absolute;
  top: 50%;                    /* 从 top: 10px 改为 50% */
  right: -12px;
  transform: translateY(-50%); /* 新增：垂直居中 */
  /* ... 其他样式保持不变 */
}
```

#### 悬停效果更新
```css
.toggle-btn:hover {
  background: #4a6cf7;
  color: white;
  border-color: #4a6cf7;
  transform: translateY(-50%) scale(1.1); /* 保持居中的同时缩放 */
}
```

#### 状态特定样式
```css
.sidebar.collapsed .toggle-btn,
.sidebar:not(.collapsed) .toggle-btn {
  right: -12px;
  top: 50%;
  transform: translateY(-50%);
}
```

## 🔧 自定义选项

如需调整按钮位置，可以修改以下CSS变量：

```css
.toggle-btn {
  top: 50%;                    /* 垂直位置：50% = 中间 */
  right: -12px;                /* 水平偏移：-12px = 突出边框 */
  transform: translateY(-50%); /* 垂直居中对齐 */
}
```

### 可选位置
- `top: 30%` - 偏上位置
- `top: 70%` - 偏下位置
- `top: 50%` - 正中间（当前设置）
